# Standard Folder

## Standard definition

Name: #DevOps

Path: X:\#DevOps

## Script

1. Check environment variable
  -  Exist
    -  System
    -  User
  -  Given folder exist
2. Check folder (exist)
1. Check wrong environment variable (exist)
  - Without prefix (#)
  - Wrong prefix; "_", "!" etc
  - Wrong casing; "devops", "Devops" etc
4. Check wrong folder (exist)
1. Warning on other drive than standard, e.g. C-drive
